# Student Marksheet Generator

A simple Java program to generate student marksheets with grade calculation.

## Features

- Takes student name, roll number, and marks for 5 subjects
- Calculates total marks, average, and percentage
- Determines grade based on percentage
- Input validation for marks (0-100 range)

## Subjects

1. English
2. Mathematics  
3. Computer Science
4. Physics
5. Chemistry

## Grading System

- 90% and above → A+
- 80% – 89% → A
- 70% – 79% → B
- 60% – 69% → C
- Below 60% → Fail

## How to Run

```bash
javac StudentMarksheet.java
java StudentMarksheet
```

## Output

The program displays a formatted marksheet with:
- Student details
- Subject-wise marks  
- Total marks and percentage
- Final grade
